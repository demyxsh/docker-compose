TIMEZONE | America/Los_Angeles

## Usage
```
docker run -t --rm \
-v /var/run/docker.sock:/var/run/docker.sock:ro \
--volumes-from=container \
--workdir="$PWD" \
demyx/docker-compose up -d
```
